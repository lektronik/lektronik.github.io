# Security Policy

## Supported Versions

Only the latest version of this project is currently supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an email to **telis.aslanidis.io@gmail.com**. All security vulnerabilities will be promptly addressed.
